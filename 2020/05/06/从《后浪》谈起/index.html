我对后浪的总结。

  1. 前浪不应过多苛责后浪，后浪不用跟随前浪的想象，双方可以一起奋斗、前行。
  2. 我看着你们，满怀：
    • 羡慕：

    • 敬意：

      • 后浪自信、大气、友善。（别夸了别夸了）

    • 感激：

正如许多文章所指出的那样，这篇4分钟左右的演讲，其实质性内容的密度比较低。

我个人觉得，作为一篇演讲来说无可厚非，毕竟，我认为《后浪》以演讲的形式进行，其目大抵可以分为：

  1. 同长辈和青年人取得共识，产生共鸣。
  2. 激励青年人奋斗。
  3. 呼吁长辈们重新审视青年。

因为其并不是说理性的文本，关注点往往是在语言的感染力上。
所以我认为，从文本的感染力上来说，《后浪》可以获得一个很高的分数。

与此同时，何冰老师对于情感的把控、词句的抑扬顿挫、恰到好处的肢体语言，不得不说，水平很高。

无论是和善的表情，还是饱满的声音，听下来都没有说教感和故作的姿态感。
相反，在他的眼神里，能看到一位长辈对一个后辈的殷切叮咛、轻声嘱托和满怀希冀。

再有，灯光、背景、视角的切换、服装等等等等，当然都是一流的，完全将观众带入到了一个舒适却不松散、严肃不失和蔼的氛围中，具有极强的感染力。

最后，我认为，视频所表现所传达的部分精神内涵值得推崇。
比如：

  • 后浪：
    1. 青年应有梦想，努力追寻想要拥有的生活。
    2. 青年应好好利用前人的成果，去探索未知的世界。
    3. 青年应注重选择，注重人际关系等等。（爱国敬业诚信友善）
    4. 青年应将青春的活力传递给世界。
  • 前浪：
    1. 正确认识后辈。
    2. 宽容理解后辈的爱好与行为。
    3. 不能将自己的想法强加于后辈身上。
    4. 鼓励、祝福后辈。

相信有相当一部分青年看过之后，热血沸腾，胸中充满了雄心壮志。
但很遗憾，我并没有体会到这样的感觉，因为我对这支视频的了解顺序，并不常规。

我了解《后浪》的历程

　　最初，一上B站，《后浪》的广告充斥着屏幕，但是我个人并不喜欢这种类型的演讲。倒不是因为我不喜欢演讲，只是他的时间实在太短了，几乎不可能有什么干货。
(个人不是很喜欢煽情类的演讲，TED倒是可以考虑考虑。)

　　所以，我最终一直没有点进去看。

　　后来，视频开始慢慢在网上发酵，群里讨论逐渐多了起来，也有同学在讨论此事。

　　我首先看到的是负面的评价，第一印象的确很重要，这导致了最初我对视频的片面理解，当时认为《后浪》是一个失败的作品，甚至，激化了社会矛盾。

　　当然，耳听为虚，眼见为实。我立即去了B站反复观看了三遍《后浪》，阅读了评论，并且观看了其“二创作品”，《前浪》，以及《非浪》。

对于《前浪》的观点

　　我更倾向于《前浪》是梗素材的无意义的剪切，纯粹为了玩梗。

　　也许作者确实想要表达什么，但以这种方式，恕我愚钝，仅仅只能看出素材本身表现的社会现象，(其中不乏有摆拍），并不能看出鲜明的内涵或者是讽刺意味。

　　想要了解《前浪》请移步《前浪》

　　（注：作者本人表示是一个娱乐作品。）

对于《非浪》的观点

　　(朱一旦老粉不请自来，吹爆！！！）（“恁奶奶个腿儿”）~~
（由于我粉丝身份的特殊性，可能包含非理性叙述，我尽量客观）

　　朱一旦的视频，一般是以一个虚构的小故事，通过反转、夸张、反复等方式，搞笑的外表下，藏着对时事热点或社会现象的观点，有些具有批判、讽刺的意味。这样将喜剧、讽刺艺术、短视频、梗一个文化等结合的作品，其实蛮少见的，这也是我喜欢导演小策团队的主要原因。

　　《非浪》便是这种艺术形式的又一力作，其几乎指明了反对《后浪》的内容。（或者说，讽刺《后浪》的意味已经快要溢出来了。）

　　“发配非洲”是朱一旦视频系列所创造的核心梗，《非浪》以其为主线，配合朱总开玩笑式的发言，和一些夸大化的社畜日常工作镜头片段，优雅不失幽默的表达了对资本压榨员工的讽刺，和对年轻人工作压力大的同情等等其他内容。

　　全站排行榜第二的《非浪》与第一的《后浪》表现了青年人完全相反的两种状态，《非浪》个中意味，不言而喻。

　　当然，《非浪》包含的，原比我以上说的要更加多，我说的也只是我个人的解读，详情移步《非浪》

视频本身所反映的问题

视频本身所反映的问题相信各位已经从互联网上了解的差不多了，我在此再啰嗦一下。

　　有人说视频更多只是B站从青年人走向全年龄段的一个广告，一个尝试，是做给前浪们看的，我比较赞同这种观点。但是，这种观点也缺少了一点自洽性，即：

  • 为什么视频要分版本？仅仅放出TV版中的演讲镜头，争议也许会更少。
  • 为什么作为五四视频发布？共青团中央、央视新闻、光明日报、中国青年报、环球时报、新京报、澎湃新闻、观察者网联合发布，视频引起的争议这么大，没人发现？

忽视青年实际收入

　　无疑，《后浪》中所使用的视频片段，在展现世界美好，青春风光无限的同时，隐藏着消费主义的影子。丰富多彩的现代文化和科技成为视频片段的绝大部分，配合精彩的剪辑专场手法，令人目不暇接。
　　
　　如果加上台词“满怀羡慕”，似乎，有点像父母的说教“你们现在生活多好啊，想想我们当时……。”
　　（我并不是反驳父母的话，只是将部分内容进行关联）

　　然而，我们真的可以支付得起视频里所表现的生活么？恐怕并不是这样的。从这篇文章（你的后浪生活，到底需要多少钱？），可以看出

  • 旅游方面，价格在四位数至七位数不等。
  • 摄影器材，价格从最低的2000左右一直增长到七位数。
  • 电脑VR设备等，万元起步。
  • 汽车，十万元起步。
  • 各种体验（跳伞啊直升机啊），300-5000。

　　看到这里，是不是有种“浪不起，浪不起”的感觉？的确，青年中大多数人是无法或是很难承担得起这些费用的，偶尔一次为了爱好咬咬牙，也是少见。

　　并不能用这些镜头去概括后浪的生活，很多时候，我们旅游、购买新电脑等等非必要的消费，我认为不能完全称之为享受生活，往往有时候我们是在对自己进行奖励。比如考试完的放松，完成业绩拿到奖金购买新手机等。

　　毕竟，如果将我们的生活比作顺流前行，这些内容就是岸边偶现的花海，水面漂浮的几点粼粼的碎金，天空不时飞过的水鸟，抛下几声清彻的鸣叫，激励我们继续前行。

　　大多数时候，只是岸边连绵不绝的青山，或是看不到边的草原，水面下湍急的暗流，炽热的阳光和如注的暴雨，我们不知道前方等待的会是什么，也许又是几道曲折的河弯，也许又是无尽的山峦和险峻的石壁，在一点一点蚕食我们对于梦想的信念。

　　不过，这条河流也并不是充满着困难和悲观的景像，相反，如果以积极的心态去面对，反倒充满着机遇与挑战，面对风雨的逆风前行，面向阳光的朝气蓬勃，欣赏两岸重复却不同的风景，与激流抗衡，与怪石周旋。

　　我们无法改变既定社会环境，但我们可以改变我们的心态。王国维在《人间词话》中写到：有我之境，以我观物，故物皆我之色彩；无我之境，以物观物，故不知何者为我，何者为物。

　　以悲观的态度看世界，阶层固化，资本剥削，感时花溅泪，恨别鸟惊心；以乐观的态度看世界，阶层流动，市场机遇，气蒸云梦泽，波撼岳阳城。

选择哪一种，是每个人的自由；适合哪一种，是每个人的特性。悲观久了，看不见世界的色彩；乐观久了，看不到社会的阴暗。如何找到一个平衡点，是一门值得研究的学问。

　　我们可以控制身下的扁舟，毕竟，河流是有宽度的。有水流湍急的中心，有水流平缓的临岸，选择紧跟时代节奏，还是歇一歇欣赏沿途的风景；在岔路时选择更适合我们的支流，是蜿蜒曲折，还是飞流直下。虽然有时迫不得已，但也只是暂时被暗流缠住。一旦渡过了黑暗，希望每个人都可以努力的将船头牢牢把握在自己手中。

　　观察收入占比，我们能够发现，绝大多数人一年内的收入几乎无法支付的起视频中所说的任何一项内容，只有约10%左右的富裕人群，才有机会享受到视频中提到的“科技繁荣、文化繁茂、城市繁华，现代文明的成果被层层打开。”

　　中国比我们想象中的要富，世界比我们想象中的要穷。

缺少五四精神

先来看一下五四精神是什么：

五四精神，是指五四运动中的精神，即：忧国忧民、热爱祖国、积极创新、探索科学的爱国主义精神。
五四精神的核心内容为“爱国、进步、民主、科学”。

五四精神表现为以下四个统一：

  1. 启蒙与救亡的自觉广泛的统一；
  2. 知识分子与劳动群众的统一；
  3. 刻苦耐劳的精神与进取创新的精神的统一；
  4. 独立自主的精神与无私奉献精神的统一。

当代爱国主义又为五四精神注入了新的精神内核与价值述求：

  1. 在国际交往日益频繁、全球化不可逆转的浪潮中，维护我们的国家利益和民族利益；
  2. 在中国发展世人瞩目的背景下，寻求我们国家在世界上全新的定位和方向；
  3. 在对外开放交往日益扩大的现实中，探讨构成我们民族精神、民族性格的新内容。

那么视频又表现了那些五四精神呢？

  • 积极创新
  • 进步
  • 科学
  • 爱国

　　显而易见，许多重要的精神被淡化了，甚至根本毫无体现。强调积极向上的一面固然没有问题，但如果以批判角度看待，难道不应该有刻苦耐劳、努力奋斗的精神么？纵观整个视频，只有一句话“奔涌吧！后浪！”。这句话更多的，应解释为前进，仍然不是强调奋斗。

　　是的，科技繁荣、文化繁茂、城市繁华，现代文明的成果被层层打开。，我们所处的时代，比以往任何一个时候都更加发达，更加自动化。可是，这并不代表我们不需要这种精神，我们不需要吃苦，不需要奋发图强。相反，我们比以往任何一个时候都更需要奋斗，更需要学会吃苦。

　　生于忧患，死于安乐。在当下，我们不能一昧的生活在父母带给我们舒适的环境，如同温室里的幼苗，溺于安详之中。困于心，衡于虑，而后作；征于色，发于声，而后喻。什么是居安思危，什么是警钟长鸣，什么是忧劳可以兴国，逸豫可以亡身，我想，每个人心里都很清楚。

中国的革命是伟大的，但革命以后的路程更长，工作更伟大，更艰苦。这一点现在就必须向党内讲明白，务必使同志们继续地保持谦虚、谨慎、不骄、不躁的作风，务必使同志们继续地保持艰苦奋斗的作风。

　　江泽民在为西柏坡题词(1991年9月21日)中写到：

我们党和军队是靠艰苦奋斗起家的，也是靠艰苦奋斗不断发展壮大起来的。今天我们搞社会主义现代化建设，同样要靠艰苦奋斗。因为搞工业现代化、农业现代化、科技现代化、国防现代化，仍然是一个需要付出百倍努力的艰苦创业的历史过程。一个国家、一个民族、一个军队，如果不提倡艰苦奋斗、艰苦创业，只想在前人创造的物质文化成果上坐享其成，满脑子是金钱万能，成天贪图享乐，那末，这样的国家、这样的民族、这样的军队，是毫无希望的，是没有不衰落和不垮台的。“历览前贤国与家，成由勤俭破由奢”，大家都要深刻地理解这个真理。

我们千万不能在一片喝彩声、赞扬声中丧失革命精神和斗志，逐渐陷入安于现状、不思进取、贪图享乐的状态，而是要牢记船到中流浪更急、人到半山路更陡，把不忘初心、牢记使命作为加强党的建设的永恒课题，作为全体党员、干部的终身课题。

　　我对物理有一定的了解，我想讲讲自然中海啸的真实原理

　　海啸是一种水波，由深水波和浅水波组成，通常由震源在海底下50千米以内、里氏震级6．5以上的海底地震引起。水下或沿岸山崩或火山爆发也可能引起海啸。在一次震动之后，震荡波在海面上以不断扩大的圆圈，传播到很远的距离，正象卵石掉进浅池里产生的波一样。

　　而形成高耸浪潮的海水，永远只是表面一层，其前行的巨大能量，来自看不见的深层海水。它们才是海啸的主体，不断地推动着海啸整体前进。由于海底泥沙岩石等的摩擦，能量逐渐耗散殆尽，在沙滩上浅浅的消散掉，你甚至不会注意到他们的存在。

视频所带来的社会问题

中国居高不下的基尼系数

首先，先援引一下百度百科，介绍一下基尼系数。

概念

　　基尼系数是指国际上通用的、用以衡量一个国家或地区居民收入差距的常用指标。

　　基尼系数最大为“1”，最小等于“0”。
　　
　　基尼系数越接近0表明收入分配越是趋向平等。

联合国开发计划署等组织规定：

  • <0.2 收入高度平均
  • 0.2 - 0.29 收入比较平均
  • 0.3 - 0.39 收入相对合理
  • 0.4 - 0.59 收入差距较大

  • 0.6 收入差距悬殊

基尼指数最早由意大利统计与社会学家Corrado Gini在1912年提出。
